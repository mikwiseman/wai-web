Frequently Asked Questions

How do I activate WaiVPN?
  1. Open the WaiVPN app.
  2. Enter your invite code on the activation screen.
  3. Tap Activate. The app will verify your code and download your VPN configuration automatically.

If activation fails, make sure your invite code is correct and that you have an internet connection. Each invite code can only be used once.

How do I connect to the VPN?

After activation, tap the Connect button on the main screen. The app will establish a secure connection to the nearest server. You may be asked to allow VPN configuration on first use — this is required by your operating system.

The status indicator will turn green when you are connected.

What if the connection fails?

If you cannot connect, try these steps:

  1. Switch servers — tap the server selector and choose a different location.
  2. Switch connection method — if available, try toggling between connection methods in Settings.
  3. Check your internet — make sure you have a working connection without the VPN.
  4. Restart the app — force-close WaiVPN and reopen it.
  5. Run diagnostics — use the built-in diagnostics tool (see below) and share the report with us.
How do I switch servers?

Tap the server name or location indicator on the main screen. A list of available servers will appear with their current latency. Tap a server to switch to it. If you are currently connected, the app will reconnect automatically through the new server.

Which platforms are supported?

WaiVPN is available on iOS, Android, and macOS. All platforms share the same account and invite code.

Built-in Diagnostics

WaiVPN includes a diagnostics tool that tests your connection and generates a report. Open the app, go to Settings → Diagnostics, and tap Run Diagnostics. You can share the report directly via email when contacting support.
